# markovian-bandwidth-law
Numerical validation and experimental reproduction for the single-window quantum bandwidth law (QBL) in Markovian open systems. MQBL-SWV.py tests linear scaling with log‑bandwidth Δ, semiclassical ε², and geometric slope K. MQBL-EV.py computes the visibility separatrix from cold‑atom data (Barontini et al.) on the same spectral ruler.
